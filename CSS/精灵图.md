# 字体图标

字体图标展示的图标，本质是文字。

字体图标的优点：

- 轻量：一个图标字体要比一些列的图像要小。一旦字体加载了，图标就会马上渲染出来，减少了服务器请求。
- 灵活性：本质其实是文字，可以随意的改变颜色，产生阴影，透明效果，旋转等。
- 兼容性：几乎支持所有的浏览器，可放心使用。

注意：字体图标不能替代精灵技术，只是对工作中图标部分技术的提升和优化。

总结：简单的图标用字体图标，复杂的图形用精灵图。
